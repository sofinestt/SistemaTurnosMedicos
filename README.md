## Introducción
El presente proyecto tiene como objetivo desarrollar un sistema de gestión de turnos médicos para un consultorio, con el fin de organizar la reserva, reprogramación y cancelación de turnos de manera eficiente. El sistema permitirá la interacción entre pacientes, secretaría y médico, incluyendo notificaciones automáticas, control de horarios, tipos de turnos y registro histórico de cambios. De esta manera, se busca mejorar la organización del consultorio, optimizar el tiempo de atención y garantizar un mejor servicio a los pacientes mediante el uso de tecnología.
