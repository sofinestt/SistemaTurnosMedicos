## Introducción

El presente proyecto es una **aplicación práctica de los principios de Diseño Orientado a Objetos (POO)** desarrollada como parte de la asignatura *Diseño Orientado a Objetos* de la Técnicatura en Programación de Sistemas.

El objetivo es construir un **Sistema de Gestión de Turnos Médicos** que demuestre cómo los fundamentos de POO —encapsulamiento, abstracción, herencia y polimorfismo— se aplican en un caso real. El sistema permitirá organizar de manera eficiente la reserva, reprogramación y cancelación de turnos, facilitando la interacción entre pacientes, secretaría y médico, e incluye funcionalidades como notificaciones automáticas, control de horarios disponibles, tipos de consulta diferenciados y registro histórico de cambios.

El proyecto utiliza metodologías ágiles de diseño como **Tarjetas CRC** (Clase-Responsabilidad-Colaborador), **diagramas de casos de uso** y **escenarios de comportamiento** para modelar la estructura del sistema de forma clara y colaborativa. Mediante este enfoque, se busca desarrollar un software que refleje la realidad del dominio, sea mantenible y extensible.
