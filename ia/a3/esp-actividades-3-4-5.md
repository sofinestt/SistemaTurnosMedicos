# Uso de IA - Especialista en Diagramas de Actividades de Casos de Uso 3, 4 y 5

## Herramienta utilizada

Se utilizó GitHub Copilot (Agent Mode) integrado en VS Code.

## Contexto proporcionado

```
Diseñe diagramas de actividades para los casos de uso 3, 4 y 5 que represente claramente flujos, decisiones y bifurcaciones, con mínimo 10 actividades y mínimo 3 swimlanes/carriles cada uno (por ejemplo, Médico, Secretaria, Sistema y Paciente).

No agregue ni cambie ningún archivo, escriba el código para cada diagrama aquí en el chat.
```

Los archivos utilizados como contexto fueron:

- diagramas/02-casos-de-uso/03-caso-uso-autorizar-sobreturno.puml
- diagramas/02-casos-de-uso/04-caso-uso-consultar-historial.puml
- diagramas/02-casos-de-uso/05-caso-uso-gestionar-disponibilidad.puml
- diagramas/03-escenarios-casos-de-uso/03-autorizar-sobreturno-autorizacion-de-sobreturno-exitoso-04.md
- diagramas/03-escenarios-casos-de-uso/03-cancelar-turno-cancelacion-de-turno-por-secretaria-existoso-02.md
- diagramas/03-escenarios-casos-de-uso/03-reprogramar-turno-reprogramacion-de-turno-exitoso-03.md
- anexos/introduccion.md

## Ajustes realizados

Se corrigieron flechas que no apuntaban a nada o no permitían un correcto entendimiento de los diagramas.

## Iteraciones

### Iteración 1

Se reformuló el prompt haciendo un breve resumen de lo que debía hacer, además de especificar que no modifique ningún archivo.

### Iteración 2

Al notar que realizó los diagramas con únicamente dos carriles, se le aclaró nuevamente y con ejemplos para que tuviera una mejor idea de qué hacer.
